# 第 5 块修订方案：格式、结构缺页、标题样式与术语统一

## 修订目标

本轮修订主要处理以下问题：

- 论文主体结构基本完整，但与首都医科大学 2022 版要求相比，仍有若干前后置页面未明确出现。
- `.docx` 的正文段落几乎全部是普通段落，真正带样式的主要是目录项，说明正文标题大概率不是标准 Word 标题样式。
- 这会直接影响后续目录、统一格式、分级标题检查和批量排版。
- 盲审和导师还指出了术语首次定义、文献上标、英文摘要格式、图题表题、重复表达等规范性问题。

## 一、目前已经确认的结构状态

从提取文本可明确看到：

- 已有：`学位论文原创性声明`
- 已有：`目录`
- 已有：`第一章` 至 `第七章`
- 已有：`文献综述`
- 已有：`文献综述参考文献`
- 已有：`附录`
- 已有：`致谢`

目前未在提取文本中明确看到：

- `学位论文使用授权声明`
- `学位论文答辩委员会组成和答辩决议`
- `攻读学位期间获得学术成果情况`
- `个人简历`

这四项里，后 3 项对博士答辩终稿尤其关键。

## 二、当前最重要的格式风险

### 1. 正文标题未样式化

通过解析 `.docx` 内部结构可见：

- 正文绝大多数段落样式为 `(none)`
- 明确出现样式的主要是 `toc 1 / toc 2 / toc 3`

这说明：

- 目录看起来是存在的
- 但正文标题本身很可能不是 `标题1 / 标题2 / 标题3`
- 当前目录可能是手工生成后遗留，或曾生成后又被破坏

直接后果：

- 目录更新不稳
- 标题字体、字号、段前段后难统一
- 后期改动后页码和层级容易错

### 2. 学校要求页面仍需补齐

首医要求中，博士论文通常还应包含：

- `使用授权声明`
- `答辩委员会组成和答辩决议`
- `攻读学位期间获得学术成果情况`
- `个人简历`

若这些内容尚未加入正文，必须在终稿阶段补齐。

### 3. 术语与写法需要全篇统一

当前已经能看到几类高频统一任务：

- `准广泛耐药肺结核` 与 `pre-XDR-TB`
- `xCT` 与 `SLC7A11/xCT`
- `柳氮磺吡啶（SASP）`
- `MDR-Chin`
- `宿主导向治疗（HDT）`

原则应当是：

- 首次出现：中文全称 + 英文全称 + 缩写
- 后续出现：统一使用一个主称呼，不要频繁切换

## 三、建议修订主线

### 1. 先补“必须有”的结构页

建议优先级：

1. `答辩委员会组成和答辩决议`
2. `攻读学位期间获得学术成果情况`
3. `个人简历`
4. `使用授权声明`

其中：

- `答辩委员会组成和答辩决议` 一般在答辩通过后定稿时加入
- `学术成果情况` 与 `个人简历` 可提前准备
- `使用授权声明` 要确认学校模板是否与原创性声明合页或单独页

### 2. 再处理标题样式

建议在 WPS/Word 中统一设置：

- 章标题 -> `标题 1`
- 二级标题 -> `标题 2`
- 三级标题 -> `标题 3`

不建议继续沿用“只靠手工加粗、改字号”的方式，因为后面目录、分页和统一排版都会受影响。

### 3. 最后处理术语、图表、参考文献和格式细节

重点包括：

- 缩写首次定义
- 文献引用上标格式
- 图题在下、表题在上
- 英文摘要格式与对齐
- 参考文献新旧比例与编号一致性
- 页眉页码、单双面打印要求对应

## 四、重点修订位置

### A. 论文前置部分

建议确认并补齐：

- 封面
- 扉页
- 原创性声明
- 使用授权声明
- 目录
- 中英文摘要

### B. 论文后置部分

建议确认并补齐：

- 文献综述参考文献
- 附录编号与标题
- 答辩委员会组成和答辩决议
- 攻读学位期间获得学术成果情况
- 致谢
- 个人简历

### C. 第七章

本章除了内容问题外，还有两个格式层风险：

- 重复编号：`7.5.1` 中出现两次“第三”
- 过度重复框架性口号，建议同步按第 4 块内容精简

## 五、建议的落地顺序

1. 先按前四块完成内容修订
2. 再在 WPS/Word 中统一标题样式
3. 更新目录
4. 补齐学校要求缺页
5. 最后逐项检查页眉、页码、图题、表题、参考文献和摘要格式

## 六、本块最终目标

让论文从“内容上已经接近定稿、但格式和结构还像工作稿”，变成：

- 内容可答辩
- 结构完整
- 样式可统一
- 能按首医终稿要求继续收尾
