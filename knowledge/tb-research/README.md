# TB Research Knowledge Base

This directory is the long-term memory for the TB research master agent.

Principles:

- Save durable learning, not just one-off summaries.
- Distinguish `FACT`, `INFERENCE`, and `HYPOTHESIS`.
- Preserve dates, links, and source provenance.
- Prefer writing reusable methods lessons, endpoint logic, bias patterns, and research gaps.
- Update targeted files rather than piling all notes into one document.

Suggested workflow:

1. Read the newest daily or weekly report.
2. Open the original paper or official source for the highest-value items.
3. Add a paper note in `paper-notes/`.
4. Update the relevant memory files in this folder.
5. Add structured provenance to `source-registry.jsonl`.

This folder should gradually make the agent more like a strong clinical research mentor than a transient literature summarizer.
