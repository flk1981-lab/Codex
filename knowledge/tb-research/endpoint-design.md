# Endpoint Design

Purpose:

- Build a reusable endpoint library for TB and adjacent clinical research.

Capture:

- microbiological endpoints
- clinical endpoints
- radiological endpoints
- patient-centered endpoints
- composite endpoint cautions
- timing windows
- relapse and recurrence logic
- treatment completion vs durable cure distinctions

For each entry:

- endpoint candidate
- best use case
- main advantage
- main failure mode
- bias or interpretability risk
