# Mentor Model

Purpose:

- Capture how the agent should think like a strong clinical research mentor.

Update this file when you learn a better way to:

- frame the real clinical decision
- prioritize among research gaps
- choose between trial, cohort, registry, or review
- critique endpoint choices
- spot avoidable bias
- balance rigor, feasibility, and publication value

Prompting rule:

- Prefer writing short mentor principles that can guide many future conversations.
